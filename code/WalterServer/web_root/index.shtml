WALTER Cerebellum





Available services are to be called via
http://server:/cortex/<command>?<params>
or
http://server:/direct/cmd=<urlencoded-command>

Commands are:
LED (on|off|blink)
ECHO hello
SETUP
ENABLE
DISABLE
POWER (on|off)
KNOB <ActuatorNo>
STEP <ActuatorNo> <incr>
CHECKSUM (on|off)
MEM (reset|list)
SET <ActuatorNo>[ ;min=<min>][ max=<max>][ null=<nullvalue>][ speed=x][ acc=x][ P=x][ D=x]
GET <ActuatorNo>[ ;n=<name>][ ang=<angle>][ min=<min>][ max=<max>][ null=<null>]
GET ll
MOVETO <angle1> <<angle2> ...<angle7> <durationMS>
LOG (setup|servo|stepper|encoder|loop) (on|off)
INFO
